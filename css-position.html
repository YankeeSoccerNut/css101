Position property has 4 options

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

Static

Static is the default for the position property.

Top, bottom, left and right will NOT work with static

This element will show up at the next available position. Nothing special and it cannot be moved.

Relative

Same as Static but it can be moved with top, bottom, left, right

If you move it, the rest of the page will think it's where it's staic postion was.
